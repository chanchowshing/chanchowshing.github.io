搭建个人静态博客
发表于2020-09-28|更新于2020-11-03|学习
|字数总计:1.3k|阅读时长:4分钟

前言

使用github pages服务搭建博客的好处：

  1. 静态文件，访问速度快
  2. 免费方便，不需要自己开服务器
  3. 数据安全，基于github的版本管理

准备工作

  • 安装git
  • 安装node.js
  • 注册、拥有github账号
    以上过程也是比较常规 简单的，可自行网上搜索相关教程

创建仓库

打开你的github主页，新建一个名为username.github.io的仓库。例如我的用户名username是chanchowshing,那么就创建chanchowshing.github.io的仓库，记住必须是你的用户名


Description是仓库描述，可以随便填写
勾选Public和Add a README file，然后点击Create Repository

绑定域名

我们可以自由选择是否绑定域名，假如没有绑定则会默认username.github.io来访问你的博客。如果你有注册自己的域名那么可以进行绑定，我个人没有绑定，具体方法请自行网上搜索

写博客

Hexo使用Markdown语法的纯文本编辑，文件后缀名.md
我推荐使用CSDN来编写博客，因为这对于新手来说非常友好
写完之后导出为Markdown文件，然后保存到根目录/source/_posts 文件夹下，例如我的是E:\hexo\source\_posts


最后打开git bash定位到根目录，执行hexo g和hexo d就完成啦
